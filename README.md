## Overall Learning 

During my 6-month Full Stack Development internship with SURE ProEd, under the guidance of Akshara Mam (Software Developer at Propel Technology), I gained strong practical experience in both frontend and backend web development. Throughout this internship, I worked on real-world projects that helped me build a deeper understanding of software development and modern web technologies.

I developed hands-on experience with technologies such as HTML, CSS, JavaScript, Node.js, MongoDB, Git, and GitHub, which helped me understand how complete web applications are designed, developed, tested, and maintained. I also improved my knowledge of version control, project management, debugging, API integration, and database handling.

Working on projects like Art Challange App gave me the opportunity to apply my technical knowledge in practical scenarios and build user-friendly, responsive applications. This internship significantly improved my problem-solving abilities, logical thinking, teamwork, communication skills, and confidence, while also helping me understand real-world software development workflows and industry standards.
